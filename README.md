# SitaIris1

Конфиг подружается через json, при отстусвии какого либо поля, есть значения по умолчанию.



Существует FTP сервер , к которому имеется доступ через пользователя "ftpuser" , "qwerty123". На сервере лежит два файла - sample.txt и filename.txt.
Приложение мониторит сервер, шифрует новые файлы по определенному пути на сервере и сохраняет их в TargetDirection, архивирует с шифрованием и потом же разархивирует и дешиврует.
